## Useful additions

- **Judge normalization**: normalize LLM outputs before scoring
- **Rubric guard**: enforce rubric-based LLM evaluation presence
- **Calibration**: ECE/Brier for confidence alignment
- **Gap scorecard**: assess requirement coverage and blind spots

## Why this matters

This is how evals mature from “task checks” into a measurement system with coverage depth and reliability.
